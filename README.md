What is “Manage Your eCommerce” (MYE)?

MYE is an e-commerce management platform: it offers multi-channel order & inventory management — meaning you can manage orders, inventory, shipping, returns, etc from more than one marketplace / sales channel (e.g. Amazon, eBay, WooCommerce, etc.). 
manageyourecommerce.com

- Features include: centralized order management; advanced inventory management (multi-warehouse or multi-location support, stock syncing across channels, automatic restocking alerts/demand forecasting) 

- Shipping & shipping-carrier integration / automated shipping / order fulfilment tools. 
- Reporting & analytics: sales trends, inventory movement, order reports, demand forecasting, profitability analysis, etc. 
- Automation features: automation rules for order routing, inventory updates, invoicing/shipping automation, etc. 
- SKU-mapping / “Smart SKU mapping” across multichannel listings (so that SKUs of the same item across different marketplaces are mapped correctly). 
- Return tracking / return-management / integrated return-tracking features. 
- Support for multi-warehouse / third-party warehouse integration, and bulk listing / updates across marketplaces. 


MYE offers a free trial option — meaning users can try before commit (which can influence test scenarios). 
Also, MYE provides standard documentation pages like Terms & Conditions, Privacy Notice, Data Processing policy, showing that they aim to comply legally with data protection requirements.
